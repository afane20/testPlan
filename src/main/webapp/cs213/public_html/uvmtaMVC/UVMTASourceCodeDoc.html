UVMTA Source Code Documentation

The following document describes the files currently use to make up the UVMTA Web Application.

Files labeled "View" are mostly PHP generated HTML and are current located in the “uvmta” directory.  These PHP files are made up mostly of HTML, CSS and JavaScript.  They make up the “view” or the major user interface code.

Files labeled "Model" interact with the MYSQL data base.  They consist mostly of database queries and interaction.

Files labeled as "model/view" are files that do database queries, but also generate html (a view) to display the records that come back from the database requests.  They are located in the “uvmta” directory.

I have also indicated how the module is accessed.  Either from a direct link or with an AJAX request.
