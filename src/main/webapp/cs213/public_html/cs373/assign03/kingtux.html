  • Areas of computer science I like the most:

    Software engineering, operating systems, and computational theory. Basically, I like doing things that if done right the user will never notice, but will have the biggest impact.

  • Areas of computer science that I hate the most:

    Web Development. I've done extensive HTML and javascript before (Started with HTML 3 then worked into HTML 4), but I can't stand them. To me, it seems like a very demeaning programming task that any script kiddy can do for minimum wage. Also, I hate how Microsoft doesn't do things the way the standards specify, which makes it a pain to build in exceptions for things that should work perfectly fine if they weren't trying to force everyone to do things in a very broken manner. Alongside that, I also don't like web development because it imposes certain limits on you as a programmer and makes me feel like I'm just playing around in the kid's pool of software development, when I could be doing some serious programming instead. Anyways, that's my little rant on web development.
